# 🔍 DataLens – Making Data Analysis Accessible

## 📌 Problem Statement

In today’s data-driven world, extracting insights from datasets still requires significant technical expertise.

Most existing tools like Tableau or Power BI are powerful, but they can be overwhelming for beginners and non-technical users.

Users often struggle with:

* Cleaning messy datasets
* Understanding data structure
* Choosing the right visualizations
* Extracting meaningful insights

This creates a gap between **data availability** and **data understanding**.

---

## 💡 Solution

**DataLens** is designed to bridge this gap.

It is an **AI-assisted data analysis dashboard** that simplifies the entire workflow:

➡️ Upload data
➡️ Clean it
➡️ Analyze it
➡️ Visualize it
➡️ Get insights

All through an intuitive and interactive interface.

---

## 🚀 Key Features

* 📂 **CSV Upload & Sample Datasets**

* 🧹 **Data Cleaning Tools**

  * Handle missing values
  * Remove duplicates
  * Drop unnecessary columns

* 📊 **Automated EDA**

  * Summary statistics
  * Correlation analysis

* 📈 **Interactive Visualizations**

  * Built using Plotly

* 🤖 **Natural Language Queries**

  * Ask questions like:
    *“Show highest sales region”*

* 🧠 **Insight Generation**

  * Identify trends and patterns

* 🔐 **Authentication System**

* 📤 **Export Cleaned Data**

---

## 🎯 Objectives

* Simplify data analysis for beginners
* Reduce dependency on complex BI tools
* Provide a guided, intuitive workflow
* Enable faster decision-making

---

## 🧠 Approach

The system follows a structured pipeline:

```
Raw Data → Cleaning → Analysis → Visualization → Insights
```

Each step is modular and interactive, ensuring users can understand and control the process.

---

## 🛠 Tech Stack

* **Backend:** Python, Flask
* **Data Processing:** Pandas
* **Visualization:** Plotly
* **Frontend:** HTML, CSS, JavaScript
* **Database:** SQLite
